# Hybrid App (FE) for connecting people in cities.

### Web Repository built using <b>Ionic 1/AngularJS 1</b>.

## Modules:

* Account<br>
* Climate<br>
* Culture<br>
* Education<br>
* Forum<br>
* General<br>
* Tax<br>
* Transportation<br>
* Visas<br>

## Environment Information:

<b>Requires</b>:

Google Maps<br>
JQuery v2.1.3<br>
Moment JS<br>
Autolinker<br>
ChartJS<br>
Ionic v1<br>
Angular Touch<br>

## Usage:

Run Ionic Server version 1 on your localhost.
